# aXure Hair Salon

A modern, responsive website for aXure Hair Salon built with React and Vite. This project showcases the salon's services, testimonials, and contact information with a clean, professional design.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Built with Tailwind CSS for a sleek, professional look
- **Interactive Components**: Smooth animations and hover effects
- **WhatsApp Integration**: Direct booking through WhatsApp
- **Service Showcase**: Detailed information about salon services
- **Customer Testimonials**: Social proof from satisfied customers
- **Contact Information**: Easy access to salon location and contact details

## Tech Stack

- **Frontend**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS 4
- **Icons**: React Icons (Remix Icon)
- **Routing**: React Router DOM
- **Linting**: ESLint

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Ultron1507/Saloon.git
   cd aure-salon
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the project for production
- `npm run preview` - Preview the production build
- `npm run lint` - Run ESLint for code quality checks
